Configuration

Fine-tune every detail

Control timing, easing, trigger point, and behaviour with data attributes.

data-aos-duration

Duration

Animation length in milliseconds. Values: 50–3000ms. Default: 600ms.

data-aos-delay

Delay

Stagger delay before the animation starts. Values: 50–3000ms. Default: 0ms.

data-aos-easing

Easing

CSS easing curve. Options: ease, ease-out-cubic, ease-out-quart, ease-out-back, etc.

data-aos-offset

Offset

Trigger offset in pixels. Higher = animates earlier. Default: 120px.

data-aos-once

Once

Animate only the first time. Set any value (e.g. data-aos-once) to enable.

data-aos-mirror

Mirror

Re-trigger animation when scrolling back up. Set any value to enable.

data-aos-anchor-placement

Anchor Placement

Defines the trigger point. Format: {el}-{viewport}. Options: top-bottom (default), center-center, bottom-bottom, top-center, etc.

data-aos-preset

Presets

Use built-in presets: subtle, smooth, dramatic, bouncy, energetic, reveal, focus, floaty.

data-aos-group

Groups

Group elements together — all members animate simultaneously when any one triggers.

data-aos-stagger

Stagger

Set on a parent to stagger child animations by N ms each. E.g. data-aos-stagger="80".

data-aos-on-start / on-end

Callbacks

Execute JavaScript functions when the animation starts or finishes via data-aos-on-start="fn()".

data-aos-animate-out

Animate Out

Remove the animation class when the element scrolls back out of view, for re-entrance effects.

Get Started

Add to your project
in seconds

Drop in two files, add an attribute, and you're done.

<!-- 1. Include CSS and JS -->
<link rel="stylesheet" href="flux.css">
<script src="flux.js"></script>

<!-- 2. Animate any element -->
<div data-aos="fade-up">
  <h1>Hello World</h1>
</div>

<!-- 3. Customize -->
<div
  data-aos="zoom-in"
  data-aos-duration="800"
  data-aos-delay="200"
  data-aos-easing="ease-out-back">
  Animated content
</div>
// Optional: init with custom global defaults
Flux.init({
  offset: 100,
  duration: 600,
  easing: 'ease-out-cubic',
  once: false,
  mirror: false,
  anchorPlacement: 'top-bottom',
  stagger: 50,
  debug: false
});
