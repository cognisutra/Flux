# Flux Animation Library

Smooth, performant CSS animations by [Cognisutra](https://github.com/cognisutra).

## Installation

Add via jsDelivr CDN:

```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/cognisutra/Flux/docs/animate.min.css" />
```

## Usage

Add the `fl-animated` class to an element, along with any animation name:

```html
<h1 class="fl-animated fl-bounce">Hello</h1>
```

## Documentation

Full documentation available on the [Flux website](https://cognisutra.github.io/Flux/).

## Accessibility

Flux supports the `prefers-reduced-motion` media query so users with motion sensitivity can opt out.
